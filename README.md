# Projeto: Busca de Livros com API (Open Library)

## Descrição

Este projeto consiste em uma aplicação web desenvolvida com **HTML, CSS e PHP** que realiza o consumo da API pública da **Open Library**, permitindo ao usuário buscar livros pelo nome e visualizar informações de forma organizada e responsiva.

A aplicação foi desenvolvida como parte da disciplina **Programação para Internet I**, com foco em integração entre front-end e back-end utilizando requisições HTTP e manipulação de dados em formato JSON.

---

## Funcionalidades

* Busca de livros por nome
* Exibição de resultados em formato de cards
* Exibição de capa do livro
* Informações como:

  * Título
  * Autor
  * Ano de publicação
* Paginação de resultados
* Tratamento de erros (falha na API, dados inválidos, busca vazia)
* Layout responsivo (desktop e mobile)
* Interface estilizada com CSS puro (sem frameworks)

---

## Tecnologias Utilizadas

* **HTML5**
* **CSS3**
* **PHP**
* **API REST (Open Library)**

---

## API Utilizada

* Open Library API
  https://openlibrary.org/developers/api

## 📁 Estrutura do Projeto

```
projeto-api/
│
├── index.html        # Página inicial com formulário de busca
├── api.php           # Consumo da API e exibição dos resultados
├── style.css         # Estilização da aplicação
└── README.md         # Documentação do projeto
```

---

## Como Executar o Projeto

1. Instale o **XAMPP** ou outro servidor local com suporte a PHP
2. Coloque a pasta do projeto dentro de:

   ```
   C:\xampp\htdocs\
   ```
3. Inicie o **Apache** no XAMPP
4. Acesse no navegador:

   ```
   http://localhost/projeto-api/
   ```
